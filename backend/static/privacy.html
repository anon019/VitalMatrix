隐私政策

最后更新：2024年11月

1. 概述

Health Assistant（以下简称"本应用"）是一个个人健康数据分析工具，旨在帮助用户分析和理解其训练数据。本隐私政策说明我们如何收集、使用和保护您的数据。

2. 数据收集

本应用通过以下方式收集数据：

  • Polar 数据：通过 Polar AccessLink API 获取您的训练记录、心率区间数据等
  • Oura 数据：通过 Oura API 获取您的睡眠、活动和准备度数据
  • 用户提供的信息：您手动输入的个人设置（如最大心率等）

3. 数据使用

我们仅将您的数据用于：

  • 计算训练指标和趋势分析
  • 生成个性化的训练建议
  • 在应用界面中展示您的健康数据

4. 数据存储与安全

  • 所有数据存储在安全的服务器上
  • 我们使用加密传输（HTTPS）保护数据安全
  • 访问令牌安全存储，不会泄露给第三方

5. 数据共享

我们不会将您的个人数据出售或共享给任何第三方。数据仅用于本应用的功能实现。

6. 第三方服务

本应用集成以下第三方服务：

  • Polar：用于获取训练数据（受 Polar 隐私政策约束）
  • Oura：用于获取睡眠和恢复数据（受 Oura 隐私政策约束）
  • AI 服务：用于生成训练建议（数据匿名化处理）

7. 用户权利

您有权：

  • 随时撤销对 Polar/Oura 的授权
  • 请求删除您的所有数据
  • 查看我们存储的关于您的数据

8. 联系方式

如有任何隐私相关问题，请通过应用内的反馈功能联系我们。

9. 政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会在应用内通知您。
